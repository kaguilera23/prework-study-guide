Prework Study Guide

HTML

  • HTML is the structure of a webpage, like the frame of a house
  • It is basic text without style or functionality
  • The header elements are generally not seen by users
  • The body elements are seen by users
  • Use as many semantic elements as possible to make the code easier to read by developers

CSS

  • An attribute, such as style, looks like style="something"
  • An element, such as style, is enclosed in open and end angle brackets
  • Three ways to add style to your HTML is with inline CSS, internal CSS Styling Sheet, and external CSS Styling Sheet
  • External CSS Styling sheets are the most popular way to add style because they're more easily editable
  • If you use an external styling sheet, you must link it in the html
  • A hexadecimal code, hex #, is a way to pick the specific color you want

Git

  • git status allows us to check which branch we're working in
  • git checkout -b feature/name tells git to create a feature branch with the name "name"
  • git pull origin main updates us with the latest modifications from the main branch
  • git add -A stages all of your changes in your local working directory
  • then use git commit -m "message" to create a commit for your changes with a message saying what the commit does
